GitHub Pages ga Kodni Joylash

Quyidagi qadamlarni bajarib, KinoUz saytini GitHub Pages orqali onlayn qilishingiz mumkin

1 GitHub Hisobi Ochish

Agar allaqachon GitHub hisobingiz bo'lmasa, quyidagi havola orqali yangi hisob oching:

GitHub Hisob Ochish

2 Yangi Repository Yaratish

GitHub da yangi repository (loyiha) yarating:

Repository nomi: kino-uz (yoki o'zingizga yoqqan nom) Description: O'zbek kinolari portali Public: ✅ belgilang Initialize with README: ✅ belgilang

3 Kodni Yuklash

Yaratilgan repository ichida "Add file" > "Upload files" tugmasini bosing va HTML faylingizni yuklang.

Fayl nomi index.html bo'lishi tavsiya etiladi.

Maslahat: Agar HTML kodini to'g'ridan-to'g'ri GitHub da yaratmoqchi bo'lsangiz, "Add file" > "Create new file" tugmasini bosing va fayl nomini "index.html" deb kiriting.

4 GitHub Pages ni Yoqish

Repository Settings > Pages bo'limiga o'ting va quyidagi sozlamalarni tanlang:

Source: Deploy from a branch Branch: main Folder: / (root) Save
